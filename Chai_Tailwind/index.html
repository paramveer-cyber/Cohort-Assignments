How It Works

01 📥
Drop the Script

Add a single script tag to your HTML. No npm, no bundler, no config. Chai boots instantly in any browser.

<script type="module"> import "chai-css-engine"; </script>
02 ✍️
Write Chai Classes

Add chai-* classes directly on your elements. Use brackets for arbitrary values — no limits.

<div class="chai-flex chai-p-4 chai-bg-[#1e1e1e] chai-radius-[12px] chai-shadow"> Hello Chai </div>
03 🎨
Styles Appear

Chai's runtime engine scans the DOM, parses every class, and injects the precise CSS rules — zero waste, zero config.

/* Auto-generated by Chai */ .chai-flex { display: flex } .chai-p-4 { padding: 1rem } .chai-shadow { box-shadow: 0 4px 24px rgba(0,0,0,0.2) }
