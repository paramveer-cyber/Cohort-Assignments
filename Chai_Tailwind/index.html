Class Patterns

1

Standalone

Keyword maps directly to a complete CSS rule. No value needed.

chai-[keyword]

2 parts · fixed output

chai-flex display: flex
chai-bold font-weight: bold
chai-center text-align: center
chai-pointer cursor: pointer
2

Prop + Value

Maps a CSS property to a value. Numbers scale by ×0.5rem unless the prop is unitless.

chai-[prop]-[value]

3 parts · dynamic output

chai-p-4 padding: 2rem (4×0.5)
chai-fs-[18px] font-size: 18px
chai-bg-[#111] background: #111
chai-z-10 z-index: 10 (unitless)

Unitless Props

These take raw numbers — no ×0.5rem scaling applied.

z opacity lh fontw fw aspect grow shrink scale flex

Arbitrary [ ] + Transforms

Wrap any value in [ ]. Underscores become spaces. Transforms always use [ ].

chai-w-[100%] width: 100%
chai-shadow-[0_4px_24px_red] 0 4px 24px red
chai-rotate-[45deg] rotate(45deg)
chai-scale-[1.2] scale(1.2)
