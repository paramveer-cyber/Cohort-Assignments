# 03 Clinic Appointment and Diagnostics Platform

This ER diagram models a clinic management system that handles patients, doctors, appointments, consultations, diagnostic tests, reports, and payments in a structured and scalable way.

I have used UUIDS as primary keys, that's why id is varchar(36) as each takes 128 bits :)
