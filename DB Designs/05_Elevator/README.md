# 05 Smart Elevator Control System

This ER diagram models a multi-building smart elevator control platform used in large commercial infrastructures such as malls, airports, corporate towers, and residential complexes. It handles buildings, floors, elevators, ride requests, allocations, maintenance, and operational logs in a structured and scalable way.

For the primary keys I have used UUIDs, which are 128 bits, therefore all IDs are stored as varchar(36).
