## Color Palette

The color system closely follows Cursor’s production design.

- Background: #14120b
- Primary Text: #edecec
- Muted Text: color-mix(in oklab, #edecec 60%, transparent)
- Card Background: #1b1913
- Hover / Elevated Cards: #201e18, #26241e
- Accent / CTA Color: #f54e00

Borders and separators use subtle low-opacity mixes of the primary foreground color.
